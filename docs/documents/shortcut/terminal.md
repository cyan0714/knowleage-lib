# 常用 Shell 快捷键

## 1. 基本操作命令
```shell
cd -
Ctrl + c # 终止当前命令
Ctrl + l # 清除屏幕
```

## 2. 快速操作命令
```shell
# 1. 移动
Ctrl + a # 光标移动到行首
Ctrl + e # 光标移动到行尾
Alt + f # 光标向前移动一个单词
Alt + b # 光标向后移动一个单词

# 2. 删除
Ctrl + u # 删除光标到行首的内容
Ctrl + k # 删除光标到行尾的内容
Ctrl + w # 删除光标前的一个单词
Ctrl + d # 向右删除一个字符
Ctrl + h # 向左删除一个字符

# 3. 其他
Ctrl + y # 插入最近删除的单词
```

## 3. 历史命令搜索
```shell
Ctrl + r # 搜索历史命令
Ctrl + p # 向前搜索历史命令
Ctrl + n # 向后搜索历史命令
```
